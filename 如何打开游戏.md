# 如何打开电梯商场

## 正常打开

1. 使用 **Cocos Creator 3.8.8**。
2. 选择项目文件夹：`/Users/jiangtianhong/Documents/电梯商场`。
3. 等待左下角资源导入完成。
4. 如果没有自动进入主场景，在资源管理器展开 `assets/scenes`。
5. 双击 `main.scene`。
6. 点击窗口顶部的三角形预览按钮运行游戏。

场景编辑器里只看到一个 `GameRoot` 节点是正常的。游戏界面由 TypeScript 在运行时创建，
需要点击预览按钮后才能看到。

## 出现空白 `scene-2d`

这表示 Cocos 打开了自动创建的临时空场景，并不代表项目内容丢失。

在资源管理器中双击：

```text
assets/scenes/main.scene
```

然后点击顶部三角形预览按钮。

## 出现 `temp/programming` 或 `ENOENT` 错误

这是 Cocos 缓存不完整。先完全退出 Cocos Creator，再删除项目中的以下目录：

```text
library
temp
```

重新打开项目后等待 Cocos 自动生成缓存，再双击 `assets/scenes/main.scene`。
删除这两个目录不会删除代码、美术、场景或存档。
